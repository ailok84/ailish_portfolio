Triangle Menu

Basic Animation

JavaScript - ProjectPool Day 1

This project was made with a fellow student using SVG animations. The idea was to create a new style of Navigation Menu. Together we came up with the idea of using triangles nested together until the user clicks on the burger menu to reveal the menu options.

We used Java Script to create the menu opening and closing and and CSS to position and animate the menu items.

The project was to be completed in one class so we focused entirely on the menu and how it functions.
