Wedding Directory

Basic UX

For the Basic UX module we had to split into groups and create a digital agency. I came up with the idea of creating a Wedding Directory where companies could sign-in and create a profile to represent their Business. Couples could then register and create a profile where they can have thousands of Wedding Suppliers at their fingertips.

We split the tasks where I decided to design and code the website. I set out making some wireframes and designing a template from there. I worked on the website with a fellow student so we worked on creating a responsive website.

We used Javascript and CSS animations to add some subtle animations to the website. The app was a group decision and we all decided how to would function and work. I really enjoyed working on this project
