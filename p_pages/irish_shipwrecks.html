We started by sketching some rough wireframes of the pages we wanted to re-design. I wanted to code a responsive menu.

Unfortunately when I merged my code with one other member of the group the website did not respond as it should and the menu didnt function as well as it should. I learned a lot from this project and it allowed me to further develop my skills in coding and design.
