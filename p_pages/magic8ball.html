Magic 8 Ball

Basic Animation

JavaScript - ProjectPool Day 2

This project was created with one other student using basic css animations and Java Script. The idea was to create a project that would use a Java Script array. We decided to use an idea of a digital Magic 8 Ball that can "predict the future".

We used Java Script to create the array which works randomely by choosing from a list of 20 possible answers

The project was to be completed in one class so I focused on the design and how it would be implemented into the code.
