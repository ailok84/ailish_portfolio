Swiss Style

Basic Web Design

Responsive Web Design

I was given Swiss Style as a theme to research and build my first responsive website grid website around. I started by researching Swiss Style and creating content for the web pages. I also designed a mood board, color swatches and a style tile to guide my through the rest of the design process. I researched break-points and decided on 3 different points.

I then began the process of designing a logo which would be suitable for the website. I had a lot of fun doing this as you can see from my image samples.

For the webpage I began creating wireframes which then progressed onto paper prototypes for Desktop Tablet and Mobile. I designed a rough web prototype in Adboe XD and did some user testing using the trunk method for the website and user experience.
